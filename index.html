Connect to Google

You need a Google Cloud OAuth Client ID. This is a one-time setup.

  1. Go to console.cloud.google.com
  2. Create a project; enable Google Drive API and Photos Library API
  3. Credentials → Create → OAuth 2.0 Client ID → Web Application
  4. Authorized JavaScript origins: add http://localhost:8000
  5. Copy the Client ID that ends in .apps.googleusercontent.com

Serve with: python3 -m http.server 8000 then open http://localhost:8000
